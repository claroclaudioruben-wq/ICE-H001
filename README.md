# ICE-H001

## Inteligencia Compuesta Evolutiva (ICE)

### Registro Público de Investigación

**Versión:** 1.0  
**Estado:** En investigación

## Objetivo

Desarrollar y validar un método para estudiar la evolución del conocimiento mediante el análisis de historiales de revisión de sistemas colaborativos.

## Caso piloto

Primer estudio sobre el historial del artículo "Gestión del conocimiento" de Wikipedia en español.

## Metodología

Cada revisión se analiza siguiendo un protocolo estandarizado que registra:

- Hechos observados.
- Tipo de cambio.
- Evidencia.
- Impacto.
- Permanencia.
- Aprendizajes.

Las conclusiones solo se consideran principios cuando están respaldadas por múltiples observaciones y evidencia consistente.

## Principios metodológicos

- La evidencia tiene prioridad sobre las opiniones.
- Las hipótesis pueden modificarse o descartarse.
- Los datos se mantienen separados de las conclusiones.
- El método debe ser replicable.
